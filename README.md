# The-Moji-Phoney-an-awesome-dumb-flip-phone-
It is a dumb phone, made of aluminium. It has a magnetic flip mechanism, and a button on the bottom to automatically open it. It has a 4Ah 15V/3A twincell LiFePO4 battery, and a 1080p top screen that shows stuff, and a 720p bottom screen. It can call, text, use an eSIM, but can't text letters- it has an extensive emoji system of Unicode + others! It runs on a custom RTOS, and can play 8-64 bit games like PacMan, Space Invaders, Snake io, Arcade Shooter. The top screen shows the actual stuff, and the bottom screen has a touchscreen for the RTOS input, as well as a physical Hall Effect joystick that pops out of the shell when it flips open. The joystick is used for scrolling, as well as moving and clicking in games. The cooling uses a porous copper plate in hotspots that leads to the outer solid aluminium case! It has a SoC. It has a small outer black and white LED screen showing time, battery percentage, and the number of the caller (if being called). You can dounload MP3s and MP4s through a USB-C, which can be played through a high quality triple-speaker system. It even has a high-quality GPS system, with multiple maps of Australia on it!
